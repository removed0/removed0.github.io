how to make a fast aternos mc server

java edition, 1.19.2
  1. server software: fabric
  2. install the lastest version of these mods:
    • alternate current
    • ferritecore
    • lithium
    • starlight
    • lazydfu (makes server startup time faster)
    • if you want you can pregenerate your world using: fabric chunk pregenerator (requires fabric api)
    • no chat reports (requires fabric api - https://saveminecraft.org)
  3. in file/server-properties, change entity broadcast range to around 80
  4. in the same file set view-distance and simulation-distance to around 5
